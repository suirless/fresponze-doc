---

The `GetDeviceFormat()` method gets endpoint device format.

## Syntax 
```cpp
void GetDeviceFormat(fr_i32 DeviceType, PcmFormat& pcmFormat);
```

## Parameters
* DeviceType - see `ETypeEndpoint` enum for more information
* pcmFormat - endpoint device format

## Return
No return.
